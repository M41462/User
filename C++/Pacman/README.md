# Pacman

A classic Pac-Man game built with C++20 and SFML.

## Build Instructions

```bash
mkdir -p build && cd build
cmake ..
cmake --build .
```

## Folder Structure

| Folder      | Description                          |
|-------------|--------------------------------------|
| `assets/`   | Fonts, sounds, and textures          |
| `include/`  | Header files (.hpp)                  |
| `src/`      | Source files (.cpp)                  |
| `build/`    | Build output directory               |
